# SoftRF MB179-SM - Heltec ESP32-S3 Port

This branch contains an experimental port of SoftRF MB179 firmware for ESP32-S3 (Heltec wireless hardware).

## Status
Development / experimental branch:
- Target: ESP32-S3 (Heltec board)
- Arduino ESP32 core 2.x (tested with 2.0.9)
- USB CDC / Serial behavior under evaluation

## Base
- Derived from MB179 stable branch
- Based on SoftRF ecosystem (Moshe Braner fork)
- Additional adaptation inspired by LY implementation

## Goals
- Port SoftRF MB179 to ESP32-S3
- Enable Heltec wireless tracker support
- Maintain compatibility with original protocol stack

## Notes
- This branch may not always compile
- Hardware-specific changes expected
- Work in progress (WIP)
