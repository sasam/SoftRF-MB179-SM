# SoftRF MB179-SM

This repository contains a fork of the SoftRF firmware based on the MB179 build.

## Status
Stable baseline firmware working on ESP32 target:
- ESP32: esp32:esp32:esp32
- Arduino ESP32 core compatible build
- Verified working configuration

## Purpose
This branch represents the **stable reference state** of the firmware used as a starting point for further hardware ports and experiments.

## Based on
- SoftRF firmware ecosystem (Moshe Braner fork)
- Adapted MB179 configuration with custom libraries

## Notes
This is the baseline version used before ESP32-S3 Heltec porting work.
